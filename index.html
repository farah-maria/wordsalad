Word Salad

This game will help you practice your understanding of the difference between a noun, an adjective, a verb and an adverb. We'll start with a few questions before building sentences with the words you have looked at.

If you need some extra help before you start, here is a reminder of the basics...

A noun is a person, place, or thing.
An adjective is a describing word. Adjectives are used to describe nouns.
A verb is a an action word! Verbs are used to describe things that nouns do.
An adverb is a word that describes a verb.

Can you see where the noun is, where the adjective is and where the verb is in the following sentence?

The black cat runs quickly.

(noun)cat  (adj)black   (verb)runs  (adv)quickly
